# **App Name**: DrawTogether

## Core Features:

- Real-time Drawing: Canvas-based drawing that syncs strokes in real-time using WebSockets.
- Multiplayer Rooms: Create and join rooms with configurable settings like max players, rounds, and custom word lists.
- Real-time Chat & Guessing: Chat system that instantly displays messages and highlights correct guesses.
- Word Selection: Drawer chooses from three randomly generated words for each round.
- Scoring System: Points awarded based on speed of correct guesses and drawer performance.
- Leaderboard: Display leaderboard at the end of each game, highlighting the winner.
- Player Nickname and roomname suggestion tool: Suggest fun, short usernames and room names using a generative AI to enhance user engagement and branding, pulling in common expressions and themes in an unpredictable fashion.

## Style Guidelines:

- Primary color: Vivid blue (#29ABE2) to evoke a sense of creativity and energy.
- Background color: Light blue (#E5F5FC) to create a clean and calming drawing environment.
- Accent color: Orange (#FF8C00) to highlight interactive elements and call-to-actions.
- Body and headline font: 'Inter', a sans-serif font, for a clean and modern look.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use simple, intuitive icons for drawing tools and settings.
- Responsive layout adapts to desktop, tablet, and mobile devices.
- Subtle animations for drawing strokes and user interface interactions.
